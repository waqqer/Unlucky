# Видео для экрана победы

## Требуемый файл

Поместите видеофайл с именем `victory.mp4` в эту папку.

## Требования к видео

- **Формат:** MP4 (H.264 codec)
- **Разрешение:** 1920x1080 (Full HD) или выше
- **Длительность:** 3-10 секунд (зацикливается)
- **Размер:** рекомендуется до 5 MB
- **Аудио:** не требуется (видео воспроизводится без звука)

## Где будет использоваться

Видео воспроизводится на фоне экрана победы при выигрыше в любой игре.

## Примеры источников

- [Pexels Videos](https://www.pexels.com/videos/)
- [Pixabay Videos](https://pixabay.com/videos/)
- [Coverr](https://coverr.co/)

## Альтернатива

Если видео не найдено, создайте заглушку или используйте CSS-анимацию вместо видео.
